# Book Store Management System


A secure REST API built using Spring Boot with:

- JWT Authentication
- Role-Based Authorization (ADMIN / CUSTOMER)
- Book Management
- Order Management
- Pagination & Search
- MySQL Database
- Swagger Documentation



###  Authentication
- Login with JWT
- Role-based access

###  Books
- Add (Admin only)
- Update (Admin only)
- Delete (Admin only)
- View all (Public)
- Search by title & author
- Pagination support

###  Orders
- Place order (Customer)
- Update order status (Admin)
- View orders

##  Tech Stack
- Java 21
- Spring Boot
- Spring Security
- JWT
- MySQL
- JPA/Hibernate
- Maven

---

##  How to Run

1. Clone repository
2. Configure `application.properties`
3. Create MySQL database `bookstore`
4. Run application
